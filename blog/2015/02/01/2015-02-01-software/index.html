软件工具

Posted on 2015-02-01 Edited on 2025-01-28 In Tech

目前手上有四套同时运行的系统：

  • Windows: 工作期间运行office，兼顾网银
  • Linux：用于各类开发
  • Mac OS X: 平时休闲
  • Android：手机系统

经常遇到很多配置的问题，这篇博客总结平时使用的各类软件工具。记录以供不时之需。

Most Powerful Softwares

  • BeyondCompare: 开发与文件管理必备
  • Sublime Text: 开发与文档必备

Mac OS X

Latex in Mac

Although I can install MacTex in a whole packet, the size is too large for me (~2G?). I choose the hardmode — install the minimal packages, and update incrementally.

I install MacTex-Basic.pkg, latexmk and Skim pdf reader, and use sublime to compile tex file. Some references are

部署MAC上的Sublime Text+LaTex中文环境

在Mac上通过Sublime、Skim编辑LaTeX

There are many problems in use. For example,

  • *.sty not found?

    • To use sudo tlmgr install *
    • If *.sty is not part of CTAN, try to find *.sty in website, and copy it to
      ‘’’
      /usr/local/texlive/2010basic/texmf-dist/tex/latex/base/
      ‘’’.
      Ref: How do I install a LaTeX .sty file on OSX?

  • Fonts not found?

    • To use sudo tlmgr install *
    • To install collection-fontsrecommended, collection-fontsextra packages;

  • eps vs. pdftex?

    • To use epstopdf package. But you need to install ghostscript first
